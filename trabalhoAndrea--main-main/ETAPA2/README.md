# OrganizaJá

Sistema de gerenciamento de tarefas acadêmicas desenvolvido para a disciplina Estruturas de Dados I.

## Como executar

```bash
python src/ui/main.py
```

## Funcionalidades

- Cadastro de tarefas
- Listagem de tarefas
- Busca linear
- Fila de pendências
- Histórico com pilha
- Desfazer ações
- Leitura de arquivo TXT
```
